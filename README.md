# Elden Ring + Git = Git Gud

Parody parody parody this is all parody please Fromsoft don't sue me!

## Features

Show a Dark Souls / Elden Ring style message when you do any of the following in VS Code:

- Commit a change
- Push a commit to your branch
- Checkout a branch
